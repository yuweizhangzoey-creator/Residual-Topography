# Residual-Topography
This repository includes five residual topography netCDF files in .nc format. These files contain residual topography calculated using different crustal thickness models and age–depth relationships.

For crustal thickness correction, three models are used: CRUST1.0 (Laske et al., 2012), ECM1 (Mooney et al., 2023), and LithoRef18 (Afonso et al., 2019).
For the age–depth relationship, three models are applied: Richards et al. (2018), GDH1 (Stein & Stein, 1992), and Crosby and McKenzie (2009).
